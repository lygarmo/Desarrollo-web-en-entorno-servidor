# Desarrollo Web en Entorno Servidor

Bienvenidos al repositorio de la asignatura **Desarrollo Web en Entorno Servidor**, donde aprendemos a crear aplicaciones web dinámicas y funcionales mediante la integración de **HTML** para el diseño y **PHP** para la lógica del lado del servidor.

## ¿De qué trata la asignatura?

Esta asignatura se centra en el desarrollo de aplicaciones web que interactúan con el servidor para procesar datos, generar contenido dinámico y ofrecer soluciones personalizadas. Aprendemos a implementar funcionalidades como formularios, sesiones y bases de datos para mejorar la experiencia del usuario.

## Tecnologías empleadas

- **HTML5**: Para estructurar y diseñar las interfaces web.
- **PHP**: El lenguaje del lado del servidor para manejar la lógica de las aplicaciones y la interacción con bases de datos.
- **MySQL**: Para gestionar datos de forma eficiente.

## Objetivos de aprendizaje

- Crear aplicaciones web dinámicas utilizando HTML y PHP.
- Procesar y validar formularios desde el lado del servidor.
- Generar contenido web dinámico en función de las entradas del usuario o datos externos.
- Integrar bases de datos (cuando sea necesario) para manejar grandes volúmenes de información.

## Proyectos realizados

En este repositorio encontrarás:

- **Formularios dinámicos**: Ejemplos de procesamiento de datos enviados desde el cliente.
- **Gestión de sesiones y cookies**: Prácticas para mantener el estado del usuario en la aplicación.
- **Generación de contenido dinámico**: Aplicaciones que producen páginas personalizadas según los datos del usuario o del servidor.

## ¿Qué encontrarás aquí?

- **Ejercicios básicos**: Para entender la interacción entre cliente y servidor.
- **Proyectos completos**: Aplicaciones web que integran HTML y PHP en escenarios del mundo real.
